# 简单工厂模式

不属于GOF 23种设计模式

## 优点

+ 只需要传入一个正确的参数，就可以获取想要的对象，而无需知道创建的细节
+ 客户端免除创建对象实例的过程，无需知道具体的细节
+ 实现了对责任的分割

## 缺点
+ 工厂类的职责太重
+ 增加新的产品，需要修改工厂的判断逻辑，违背开闭原则
